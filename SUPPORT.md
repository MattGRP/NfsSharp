# Support

Use [GitHub Issues](https://github.com/MattGRP/NfsSharp/issues) for reproducible bugs and focused feature requests.

Include the following information where relevant:

- NfsSharp package and version.
- Target framework and operating system.
- NFS server implementation and version.
- Export and authentication mode, with sensitive values removed.
- Exception details and a minimal reproduction.

For usage questions, start a GitHub Discussion if discussions are enabled. Security reports must follow [SECURITY.md](SECURITY.md) and should not be filed publicly.
